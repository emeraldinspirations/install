# Install Applciation
```bash
sudo apt-get install i3 fonts-font-awesome compton feh i3lock scrot imagemagick xautolock rofi xclip
```

# Prep for settings
```bash
mkdir -p ~/.i3
```

# Add Symbolic Links
```bash
ln -s ~/Install/i3/.i3status.conf ~/.i3status.conf
ln -s ~/Install/i3/config_ ~/.config/i3/config
```
